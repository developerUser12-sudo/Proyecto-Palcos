# Proyecto-Palcos

Proyecto personal desarrollado para seguir aprendiendo y experimentar con nuevas tecnologías. Consiste en una aplicación web que permite reservar palcos de la Carrera Oficial de la Semana Santa de Utrera por días, facilitando una gestión más flexible que el sistema de reserva semanal actual.

Una vez finalizado el desarrollo, el proyecto será presentado al Consejo de Hermandades y Cofradías de Utrera como propuesta de mejora del sistema de reservas.

## Tecnologías

- Spring Boot
- Java
- PostgreSQL
